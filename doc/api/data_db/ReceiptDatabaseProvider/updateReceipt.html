updateReceipt method

Future<int> updateReceipt (Receipt receipt)

Update Receipt

Use this method to update a receipt in the database.

To update a receipt, the receipt id in the argued receipt object must match the id of the desired receipt to update. The target receipt in the database will be updated to the exact values of the values of the argued receipt object, inclusive of nulls.

Example

For an existing receipt { "id" : 101, "total" : 2800, "receiptDate" : 89000 }

When updateReceipt is called with the argument { "id" : 101, "total" 2799 }

The updated receipt in the database will be { "id" : 101, "total" : 2799, "receiptDate" : null }

Implementation

Future<int> updateReceipt(Receipt receipt) async {
  final db = await database;
  return await db.update(
    table,
    receipt.toMap(),
    where: "$id = ?",
    whereArgs: [receipt.id],
  );
}
