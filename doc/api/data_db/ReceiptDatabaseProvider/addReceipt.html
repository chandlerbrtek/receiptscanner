addReceipt method

Future<Receipt> addReceipt (Receipt receipt)

Add Receipt

Use this method to add a new receipt to the database.

When adding a receipt, the database will automatically assign a new id for the receipt. This id is returned on the response receipt object.

Note: If an id is specified and another receipt with the same id is specified within the database, that receipt will be overwritten by the new receipt. The update method should be used in such a scenario.

Example

addReceipt ( { "total" : 508, "receiptDate" : 15003 })

Returns the receipt

{ "id" : 37872, "total" : 508, "receiptDate" : 15003 }

Where the id is the next available id in the database.

Implementation

Future<Receipt> addReceipt(Receipt receipt) async {
  final db = await database;
  receipt.id = await db.insert(
    table,
    receipt.toMap(),
    conflictAlgorithm: ConflictAlgorithm.replace,
  );
  return receipt;
}
